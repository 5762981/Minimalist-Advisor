# Minimalist AI Advisor
이 레포지토리는 감정적 미련으로 버리지 못하는 물건들의 처분을 돕는 '미니멀리스트 AI 물건 버리기 제안 서비스'의 소스 코드와 기획 문서를 포함하고 있습니다.

## 서비스 개요
해당 서비스는 사용자의 공간 기회비용을 분석하고, AI 객관화 모델을 통해 물건의 처분 여부(유지/폐기)와 구체적인 행동 가이드를 제안하는 AI 코칭 플랫폼입니다.

## 주요 기능
* 객관적인 AI 판결 모드: OpenAI(gpt-4o-mini)를 활용해 냉철한 폐기를 유도하거나, 필수품의 경우 '현상 유지'를 제안
* 공간 손실 환산기: 물건이 차지하는 부피를 월세로 환산하여 매달 낭비되는 주거 비용 안내
* 맞춤형 행동 계획 제안: 당근마켓 중고 거래, 폐기물 처리 등 구체적인 처분 방법 가이드 제공

## 상세 기획서 (PDF)
서비스 기획 배경, 시스템 아키텍처, 향후 멀티모달(Vision AI) 고도화 방안 등 프로젝트에 대한 상세한 내용은 첨부된 `AI 스타트업 프로젝트 과제 기획서_계명대 곽윤경.pdf` 슬라이드를 참고해 주세요.

## 로컬 실행 방법
# 1. 라이브러리 설치
pip install -r requirements.txt

# 2. 최상단 디렉토리에 .env 파일 생성 후 API 키 입력
OPENAI_API_KEY=sk-본인의API키

# 3. Streamlit 앱 실행
streamlit run main.py
